## Lemon Squeezy

Ready to make the jump from Lemon Squeezy to Spaire? Use the `spaire-migrate` CLI tool to quickly and easily migrate your existing Lemon Squeezy products to Spaire.

### Getting Started

```bash Terminal
npx spaire-migrate@latest
```

### Supported Migrations

* Products & Variants
* License Keys
* Associated Files
* Discount Codes
* Customers

This tool is not able to move **active** subscriptions from your Lemon Squeezy store.
